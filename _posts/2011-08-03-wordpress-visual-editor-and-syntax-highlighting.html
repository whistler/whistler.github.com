--- layout: post title: Wordpress Visual Editor and Syntax Highlighting tags: - code - editor - highlight - syntax - Wordpress status: publish type: post published: true meta: _edit_last: "1" --- Somehow the visual editor was not working on my wordpress installtion. I checked the server log and it turned out that the file permissions were not set up correctly. Just needed to give do chmod -R +r . in the wp-includes/js directory to get it working again. Also the previous syntax highlighter I had installed sucked. I tried out a couple and so far I only like CodeColorer I also really like the syntax highlighter on this website but I dont know if that is available on wordpress.
